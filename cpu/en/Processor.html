What is a processor?

A computer's microprocessor is its Central Processing Unit (CPU). It is the core or brain of the computer, it performs basic calculations and manages the operation of the rest of the components. It consists of the Control Unit (CU) and the Arithmetic Logic Unit (ALU).

The processor is an electronic circuit and works by performing basic arithmetic operations (addition, subtraction, multiplication and division), logical operations (logic gates, AND, OR, XOR...) and control operations (synchronization clock with the main memory), and is responsible for processing the computer's input and output data ( assisted by the PCH (Platform Controller Hub) to control the peripherals).

Processor cycles

Processors perform millions of operations per second. However, a processor is only capable of performing a very limited number of operations per cycle. In contrast, today's processors are capable of millions of cycles per second.

A CPU cycle is an electromagnetic pulse. This pulse is emitted by an internal clock of the processor. Each time this oscillator emits a pulse the processor performs a series of operations. Processor cycles are measured in GHz.

Processor architecture

The processor is the coordinator of all the tasks performed by a PC, which means that its internal design is very important. Inside, it is nothing more than a set of interconnected blocks in which each one of them performs a function. The design of these elements and how they are interconnected is what is called processor architecture.

Control Unit (CU)

It is in charge of performing the instruction cycle:

  • Fetch: read the instructions in memory (i.e. the program to be executed).
  • Decode: decode the instructions and, if necessary, the memory addresses involved.
  • Execute: execute the instructions given by the program (sending the necessary commands to the rest of the processor components).​
​

Arithmetic Logic Unit (ALU)

It is responsible for performing arithmetic (addition, subtraction...) and logical (NOT, AND, OR, XOR...) operations with the received data. It consists of a series of modules that perform the different operations. The CU is in charge of selecting the operation to be performed by enabling the data paths between the various ALU operators and between the internal registers.

Cache

They are an intermediate memory between the main memory (RAM) and the processor. They are faster but smaller than the main memory. They are hierarchically divided into L1, L2, L3, or more, with L1 being the fastest and smallest and L3 or L4 or... the slowest and largest.

Registers

These are small, very fast memory units located inside the CPU. They store the results of instruction execution and the internal CPU configuration or information during the last ALU operation. There are different registers with different functions, some of the most important ones are:

  • Program counter: Stores the address of the next instruction to be executed.​
  • Instruction Register: stores the instruction captured in memory, the one that is being executed.​
  • Status Register: reports the result obtained in the last ALU operation.​
  • Accumulator Register: stores the results of arithmetic and logical operations.

External connections

One of the big steps that made computers possible was the incorporation of an internal memory (the main memory) that allowed computers to store not only the data but also the program to be executed.

Modern computers have many external connections. Connections to the main memory (RAM), and to a large number of input and output devices: hard disks that allow permanent storage of data and programs, keyboard, mouse, GPU, audio inputs and outputs...

Other processor types

So far we have referred interchangeably to processor or CPU, however, they are not the same from a specialized point of view.

Technically a processor is an integrated electronic circuit that performs logical, arithmetic and input/output operations. Under this definition the CPU is only one of the processors in a current computer. It is the processor that is dedicated to all general tasks that do not have a specialized processor. Another processor found in all computers is the GPU (Graphics Processing Unit).

This is a list of some of the processors that exist:

  • Graphics Processing Unit (GPU): GPUs were introduced at the end of the last century due to the need to perform graphics calculations quickly and thus keep computers from being laggy when they had to render more complex animations.
  • Application-Specific Instruction-set Processor (ASIP): ASIP processors are those that have been created to perform a single function. Their circuitry is specially designed for a single type of computation.
  • Physics Processing Unit (PPU): A PPU is a microprocessor designed for physics computation, especially for the physics engine of a video game.
  • Digital Signal Processor (DSP): A DSP is a specialized microprocessor whose architecture is optimized for the needs of digital signal processing.
  • Cuantum computers: Quantum computers, being a new paradigm at both the hardware and software levels, have their own processors and technology very different from what we have seen so far.
