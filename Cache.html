What is a cache?

Cache memory is a very fast type of memory that contains a small set of instructions that your computer uses regularly to perform its daily tasks. The computer loads these instructions into the cache using complex algorithms in order to access them quickly, efficiently and without further obstructions, which greatly increases efficiency.

Functioning

Cache memory does not work in the same way as RAM memory, we cannot make the CPU address a specific address within the cache. This is because what the cache does is copy the data that is in RAM and is close to the memory address that the CPU is currently executing, so the cache copies the memory addresses closest to the cache memory inside it. Proportionally speaking, nowadays it is common to handle capacities of several terabytes when we talk about hard disks, while when we talk about RAM, this amount is reduced to a few gigabytes (nowadays the most common is to see 16 or 8 GB of RAM). Cache, on the other hand, is even smaller, and is actually measured in megabytes or kilobytes.

Levels

The cache is divided into a total of three general levels to which we can add a fourth, which is not at all common and which plays a particular role: L1, L2, L3 and L4. The difference between L1, L2 and L3 cache memory is based on a hierarchy of proximity to the processor, speed and capacity.

Cache L1

It is the most basic level, the closest to the processor and the fastest. It also has the lowest capacity, for example a two-core processor such as the Pentium G4560 has 64 KB in total (32 KB per core), although the most powerful models (Xeon and Epyc series) can reach 3 MB without any problem.

Cache L2

It is an intermediate level that presents a good balance between capacity, closeness and speed. Following the example above, the Pentium G4560 has 512 KB (256 KB per core) of L2 cache, but a Threadripper 2990WX comes to 16 MB in total (512 KB per core).

Cache L3

It ranks lower than the former in both closeness and speed, but has a much higher capacity. The Penitum G4560 has 3MB of L3 cache shared across its two cores, while the Threadripper 2990WX comes with a total of 64MB (16MB shared per eight-core block).

Cache L4

This is an unusual type of cache memory that is typically used to support performance enhancement of integrated GPUs. For example, the Core i5 5775C came with 6 MB of L3 cache and 128 MB of eDRAM as L4 cache, which was used as a buffer for the integrated Intel Iris Pro 6200 graphics card. This improved bandwidth and reduced the impact of having to use RAM as graphics memory.
